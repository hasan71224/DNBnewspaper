Add a simple and beautiful author box with social media profile buttons after your posts and pages

== Description ==
* Add a simple, beautiful and responsive author box below your posts and pages
* Use the [socialpug_author_box] to add the author box anywhere in your template files
* Add social network profile buttons for each of the authors
* Sort the social buttons to showcase certain social networks for each author


= Documentation =

http://docs.devpups.com/social-pug/author-box/

= Website =

http://www.devpups.com/

== Installation ==

1. Upload the social-pug-author-box folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Activate each location where you wish the buttons to appear
